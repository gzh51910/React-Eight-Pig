

# 8只小猪

8只小猪由一群热爱旅游的小伙伴，结合自身痛点开发的APP。我们跨越地域，连接全世界会玩的达人和渴望深度体验的游客，让你无论去哪个国家，都能找到最懂本地吃喝玩乐的达人，像朋友一样带你尽情游玩。
